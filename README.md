# psychological-assessment-system

A Streamlit-based automated psychological assessment scoring system developed using Python.

## Features
- TAS-20 automatic scoring
- Reverse scoring support
- BEIS-10 scoring
- DASS Anxiety scoring
- Automated interpretation generation
- SQLite database integration
- Search and filtering
- CSV/XLSX export support
- Male/Female categorization

## Technologies Used
- Python
- Streamlit
- Pandas
- SQLite

## Workflow
Raw questionnaire responses → Automated scoring → Interpretation generation → Database management dashboard
